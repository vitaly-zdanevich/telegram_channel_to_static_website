My new {{ tag(t="bash") }} {{ tag(t="alias") }}

```
bind '"\ej": "!#:$ \e^"'
# Alt J repeats prev word
# This is useful when, for example, you want to rename a file,
# for example - to adjust it file extension.
```

`~/.inputrc` syntax is simpler:

```
"\ej": "!#:$ \e^"
```
